clickListener()

It checks if user clicks on the lock-icon which is located in the password field. If the lock-icon is clicked, confirmPasswordVisibility() function is called. If somewhere else in page is clicked, but not on lock-icon, it ensures that the password is no longer visible, and shows back the lock-icon.
Parameters:
Name Type Description

confirmInput.type string If type is "password", then the password is not visible, if type is "text" then it's visible.
confirmIcon.src icon URL of matching icon for each input type.

isEmailValid(email) → {boolean}

Validates an email address according to custom rules. The rules are: - Email must have just one '@' character. - '@' splits 'email' into 'name part' and 'domain parts'. - Name part must have at least 8 characters. - Name part must contain at least 6 letters. - Name part must contain no more than 2 numbers. - '.' splits domain parts into provider and domain. - Provider must be valid and from the list. - Domain must contain just letters and at least 2 of them
Parameters:
Name Type Description
email string The email address to validate.
